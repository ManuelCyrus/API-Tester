# API Tester

Uma extensão para o Google Chrome que permite testar endpoints de APIs REST diretamente no navegador. Suporte a métodos `GET`, `POST`, `PUT`, `DELETE`, com headers, body e resposta formatada.


---

##  Funcionalidades

- Inserção de URL, headers e corpo da requisição (JSON ou form-data)
- Suporte a `GET`, `POST`, `PUT`, `DELETE`
- Visualização da resposta com destaque para JSON
- Status HTTP retornado
- Interface compacta e responsiva
